<h1 align="center">SAN-PLIS</h1>

<p align="center">
Personal Ledger Intelligence System web application · Built in 2024<br>
<b>My first full-stack project</b>
</p>

<hr>

<h3>Tech Stack</h3>

<b>Frontend</b><br>
HTML · CSS · JavaScript · Particles.js<br><br>

<b>Backend</b><br>
Java · JSP (Servlet Pages) · Apache Tomcat<br><br>

<b>Database</b><br>
MySQL · JDBC (MySQL Connector)

<hr>

<h3>Features</h3>

• Ingest new transactions – add expenses seamlessly<br>
• Purge transactions – remove outdated or erroneous entries<br>
• Query transaction history – view expenses with latest-first ordering<br>
• Persistent relational storage – backed by SQL for reliability and data integrity

<hr>

<h3>Deployment(For Testing Purposes)</h3>

XAMPP · Apache Tomcat · MySQL Connector (.jar)<br>
Deployed locally via Tomcat <code>webapps</code>

<hr>

<h3>Purpose</h3>

Built to understand backend logic, database integration,  
JSP & Servlets, and local server deployment.
